## üîç HOW IT WORKS

### The 7 Signals

1. **Volume Spike** - Unusual activity (institutional accumulation?)
2. **Price Consolidation** - Base building before breakout
3. **Technical Breakout** - Price breaks key levels
4. **Catalyst Events** - Earnings, FDA approvals, insider buying
5. **Sentiment Shift** - News sentiment changing
6. **Insider Activity** - Directors/officers buying
7. **Pivotal Points** - Livermore's 100-year-old pattern (still works)

### Convergence Score

When multiple signals align on the same stock = HIGH CONVERGENCE

```
1-2 signals = 30-50 (ignore)
3-4 signals = 60-75 (watch)
5+ signals = 85-95 (ACT)
```

### The Filters

**Before EVERY trade:**
1. **Learner Filter** - Does this match patterns that worked before?
2. **10 Commandments** - Does this pass Market Wizards' rules?
3. **Risk Management** - Max 2% risk per trade (Kovner's law)
4. **5:1 R/R Minimum** - PTJ's rule (can be wrong 80% of time)

### The Learning Loop

```
Trade ‚Üí Record Outcome ‚Üí Analyze Pattern ‚Üí Extract Rules ‚Üí Apply to Future Trades
```

**What it learns:**
- When to cut losers (drawdown patterns)
- What convergence scores actually work
- What volume confirmation matters
- What consolidation time is enough
- Self-heals by not repeating mistakes
